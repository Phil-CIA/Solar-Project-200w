# Solar Project 200W

Initial workspace for the 200W solar project.

## Project Goals
- Define system requirements and constraints
- Develop hardware and firmware iteratively
- Track testing, calibration, and safety checks

## Repository Structure
- `docs/` project documents, requirements, test plans, logs
- `hardware/` schematics, PCB files, BOM, manufacturing outputs
- `firmware/` embedded code and related tooling

## Getting Started
1. Add project requirements in `docs/requirements.md`.
2. Capture architecture notes in `docs/architecture.md`.
3. Add firmware target details in `firmware/README.md`.
4. Add hardware toolchain details in `hardware/README.md`.
